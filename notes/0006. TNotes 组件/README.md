## 2. 🫧 评价

这一章节中介绍的这些组件是 TNotes 中封装的一系列 `.vue` 模块，本章只记录这些组件如何使用，具体实现可以到 `.vitepress/components/*` 中查看相关模块源码。

本章节下的笔记除了对核心组件的使用做一个简单的介绍之外，还有另一个作用：开发阶段，测试组件是否能够正常工作。
