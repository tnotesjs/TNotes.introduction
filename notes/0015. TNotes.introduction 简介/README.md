## 1. 🎯 本节内容

- TNotes.introduction 知识库简介

## 2. 🫧 评价

`TNotes.introduction` 知识库是基于 `TNotes` 搭建的一个开源知识库，顾名思义，（introduction）这个知识库就是用来介绍和 `TNotes` 这款工具相关内容的，可以将它视作 `TNotes` 的使用说明文档。

## 3. 🤔 TNotes.introduction 这个知识库主要记录些什么内容呢？

1. TNotes 核心组件的基本使用说明 + 功能演示
2. TNotes 核心命令的基本使用说明
3. TNotes 知识库配置文件和笔记配置文件的介绍
4. TNotes 架构以实现原理简介
5. TNotes 的一些迭代 Log
6. ……
