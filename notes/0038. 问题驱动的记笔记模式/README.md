## 2. 🫧 评价

个人的书写习惯：以问题驱动的形式来理清思路，整理笔记！

这种方式在当前（26.03）这个 AI 时代感觉还是挺好的，顺带还能锻炼我们的提问方式。

缺点也比较明显，就是标题过长！

好在问句形式的长标题通常都是二级标题，看习惯了也就还能接受，每个二级标题都记录一个具体的内容，可以根据目录中这篇笔记的二级标题来快速了解大概都记录了哪些内容。
